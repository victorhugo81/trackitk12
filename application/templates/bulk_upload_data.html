info Instructions

  1. Download the CSV template using the button on the upload form.
  2. Fill in all required columns for each user row. Do not remove or rename column headers.
  3. If a user with the same email already exists, their record will be updated (room, role, and site). If not, a new user will be created.
  4. Any active user not found in the uploaded file will be automatically marked Inactive.
  5. New users are assigned a random temporary password and will be required to change it on first login.
  6. Save your file as filename.csv (UTF-8 encoding) before uploading.
